# Taskz

Taskz is a command line Ruby gem that helps you to remember things you wanted to do. 

### Version
0.0.1

### Installation


You need a working version of Ruby installed.

```sh
$ gem install taskz
```

### Usage

Displays help:
```sh
$ taskz --help
```

Displays tasks:
```sh
$ taskz
```

Add a task:
```sh
$ taskz -n "My new Task!"
```
